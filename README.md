# Pytest home work


### Требуется:

● Написать параметризированный тест с помощью PyTest, который проверяет,<br>
что входной список email является корректными email.<br>
Пример корректных email 'test@test.ru', 'w@w.com', '123QWE@mmm.mmm’.

● Написать параметризированный тест с помощью PyTest, который проверяет,<br>
что входной список email является некорректными email.<br>
Пример некорректных email 'test@test.', 'w@', '@tt’.

● Сделать фикстуру, которая передает имя файла,<br>
в который будут писаться результаты тестов (например 1 или 0, на ваше усмотрение).

● Имя файла задается в ini файле, дефолтное значение ‘log.txt’.
